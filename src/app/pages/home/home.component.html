ngx-foundation

An Angular Component Library
for Foundation for Sites.

Requires Angular 6 or 7 and Foundation 6.4 or later.
A fork of the awesome ngx-bootstrap by Valor Software.

Get Started Github


Current Version: 1.0.6

Less Code Less Work

Simplified code with clear examples and a starter project to get your components up and running in no time with little effort. Only import the components and styles that your project needs to keep it lightweight.

No jQuery or Additional Libraries

No need to import jQuery or additional external libraries that can add weight to your application. ngx-foundation utilizes the power of Angular and the speed of Foundation to make your app fast.

Flexible Code and Styling

Implement your own templates and use your own custom Foundation styles. ngx-foundation was made with flexibility in mind with a minimal amount of added selectors.

Responsive and Mobile-Ready

You can expect ngx-foundation to perform with speed and efficiency whether you're on a desktop or mobile device. ngx-foundation uses ZURB's Foundation for Sites which is the most advanced responsive front-end framework in the world.
